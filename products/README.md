# products/ — 商品フォルダの置き方

商品ごとに1フォルダ。フォルダ名は英数字ケバブケース推奨（例: `soup-curry-nanairo`）。

```
products/<商品名>/
├─ images/            # 商品写真を放り込む（png/jpg/webp）
└─ product-info.md    # 自由記述メモ
```

## product-info.md は自由記述

決まったフォーマットはない。商品名・価格・特徴・フレーバー・内容量・販売期間・
世界観の希望・NGな方向性など、伝えたいことを普通に書けばよい。
フライヤーに最低限必要なのは「商品名・価格・買える場所or期間」程度。

## images/ のコツ

- しずる写真（美味しさが伝わる一枚）があると強い：自然光・湯気・断面・スタイリングあり等
- 透過PNG（背景除去済み）と白背景写真は両方使える（扱いが変わるだけ）
- ロゴ・エンブレムの画像や参考資料もここに入れてよい（ファイル名で分かるように）
