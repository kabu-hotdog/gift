# references/ — デザイン参考資料

`products/` を横断して使う、gift全体共通の参考資料置き場。特定商品専用の資料ではなく、
デザインの引き出しを増やすための素材はここに入れる。

## 何を置く場所か

- 競合・他社の食品ギフトフライヤー・パッケージ画像（構図・配色・訴求の参考）
- 気になったポスター・広告のスクリーンショット（食品に限らず、構図やタイポの参考になるもの）
- テーマ発想のメモ（「のし」「お歳暮カタログ」等、ギフト文脈の記号を調べた結果など）
- ブランドガイドライン・ロゴ規定など、複数商品にまたがって関わる資料

## 使い方

- ファイル名で内容が分かるようにする（例: `2026-competitor-yubari-melon-box.jpg`）
- サブフォルダを切ってよい（例: `references/competitors/`, `references/typography/`）
- 生成前のリサーチで見つけた良い参考例は、思い出せるようにここに集めておく
  （`.claude/skills/context-framework.md` §2「リサーチ・ファースト」の実践場所）

商品固有の資料（パッケージ写真・ロゴ素材など）は `products/<商品名>/images/` に置く。
